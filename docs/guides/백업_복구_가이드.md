# 백업 복구 가이드

## 백업 상태 확인
✅ Supabase 백업이 확인되었습니다.

## 복구 방법

### 방법 1: Supabase 대시보드에서 직접 복구 (권장)

1. Supabase 대시보드 → Database → Backups
2. **2026-01-31 16:00:14** 백업 선택 (또는 2026-02-01 백업)
3. "Restore" 버튼 클릭
4. 복구할 시점 선택
5. 복구 실행

⚠️ **주의**: 전체 데이터베이스가 복구되므로, 최근 작업(2026-02-02 이후)이 사라질 수 있습니다.

### 방법 2: 백업 파일 다운로드 후 선택적 복구

1. 백업 파일 다운로드
2. 백업 파일에서 `event_survey_entries` 테이블 데이터 추출
3. survey_no 91~174 범위 데이터만 선택적으로 복구

## 복구 후 확인 사항

1. event_survey_entries 테이블에서 survey_no 1~177 범위 확인
2. 더미 데이터와 실제 데이터 구분 확인
3. marketing_stats_daily와 일치 확인
