# 도메인 변경 공표

**공표일**: 2026-01-28  
**적용일**: 즉시

---

## 📢 공표 내용

**`must.ai.kr` 도메인은 더 이상 사용하지 않습니다.**

모든 링크와 URL은 **`https://eventflow.kr`** 도메인을 사용합니다.

---

## 🔄 변경 사항

### 1. 기본 도메인 변경

- **이전**: `https://must.ai.kr` (사용 중단)
- **현재**: `https://eventflow.kr` (공식 도메인)

### 2. 영향 범위

다음 기능에서 `eventflow.kr` 도메인을 사용합니다:

- ✅ 캠페인 링크 생성 (UTM + cid 포함)
- ✅ 웨비나 짧은 링크 생성
- ✅ 클라이언트 초대 링크
- ✅ 모든 공개 이벤트 페이지
- ✅ API 응답의 URL 필드

### 3. 코드 변경 사항

모든 API에서 기본 URL은 다음과 같이 설정됩니다:

```typescript
const baseUrl = process.env.NEXT_PUBLIC_APP_URL || 'https://eventflow.kr'
```

**환경 변수 설정:**
- `NEXT_PUBLIC_APP_URL` 환경 변수가 설정되어 있으면 해당 값 사용
- 설정되지 않은 경우 기본값 `https://eventflow.kr` 사용

---

## 📋 마이그레이션 가이드

### 기존 `must.ai.kr` 링크 사용자

기존에 `must.ai.kr` 도메인으로 생성된 링크는:

1. **자동 리다이렉트** (구현 예정)
   - `must.ai.kr` → `eventflow.kr`로 자동 리다이렉트

2. **수동 업데이트 권장**
   - 새로운 링크는 `eventflow.kr`로 생성
   - 기존 링크도 가능하면 `eventflow.kr`로 교체

### 새 링크 생성

모든 새 링크는 자동으로 `eventflow.kr` 도메인으로 생성됩니다:

```
https://eventflow.kr/event/149403/register?cid=X5L2G9KV&utm_source=google&utm_medium=cpc&utm_campaign=...
```

## 📝 참고 사항

### 명세서 반영

`docs/광고캠페인_UTM` 명세서 Section 7.1에 명시된 대로:

> **항상 `https://eventflow.kr` 사용 (must.ai.kr 사용 금지)**

이 원칙이 코드에 반영되었습니다.

### 환경 변수 설정

프로덕션 환경에서는 `.env.local` 또는 Vercel 환경 변수에 다음을 설정하세요:

```env
NEXT_PUBLIC_APP_URL=https://eventflow.kr
```

로컬 개발 환경에서는:

```env
NEXT_PUBLIC_APP_URL=http://localhost:3000
```
