# Supabase 백업 복구 가이드

## 백업 목록

- 2026-02-01 16:07:10 (UTC) = 한국시간 2026-02-02 01:07:10
- 2026-01-31 16:00:14 (UTC) = 한국시간 2026-02-01 01:00:14 ⭐ **권장**

## 복구 단계

### 1단계: Supabase 대시보드에서 백업 복구

1. https://supabase.com/dashboard 접속
2. 프로젝트 선택
3. 왼쪽 메뉴: **Database** → **Backups**
4. **2026-01-31 16:00:14** 백업 선택 (또는 2026-02-01 백업)
5. **"Restore"** 버튼 클릭
6. 복구 확인

⚠️ **주의사항:**
- 전체 데이터베이스가 복구됩니다
- 2026-02-02 이후 작업이 사라질 수 있습니다
- 복구 전에 현재 상태를 백업하세요

### 2단계: 복구 후 확인

복구가 완료되면 다음 명령어로 확인:

```bash
npx tsx scripts/restore-from-backup.ts
```

### 3단계: 복구 후 작업

1. 더미 데이터 정리 (필요시)
2. marketing_stats_daily 확인
3. 참여자 목록 확인
