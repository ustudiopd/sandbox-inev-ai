# 데이터 복구 가이드

## 문제 상황
- survey_no 91~174 범위의 실제 참여자 데이터 84개가 삭제됨
- 현재 실제 데이터: 93개 (survey_no 1~90, 175~177)

## 복구 방법

### 방법 1: Supabase Point-in-Time Recovery (권장)

1. Supabase 대시보드 접속
2. 프로젝트 선택
3. Database → Backups 메뉴
4. Point-in-Time Recovery 활성화 확인
5. 삭제 전 시점 선택 (예: 2026-02-02 오전 시간대)
6. 해당 시점으로 복구

### 방법 2: Supabase SQL 백업 복구

1. Supabase 대시보드 → Database → Backups
2. 최근 백업 파일 다운로드
3. 백업 파일에서 `event_survey_entries` 테이블 데이터 확인
4. survey_no 91~174 범위 데이터 추출
5. 현재 데이터베이스에 INSERT

### 방법 3: 수동 복구 (백업이 없는 경우)

1. CSV/Excel 파일에서 데이터 확인
2. 수동으로 데이터 입력
3. 또는 참여자에게 재등록 요청

## 복구 후 확인 사항

- survey_no 1~177 범위의 모든 데이터가 있는지 확인
- 더미 데이터와 실제 데이터가 섞이지 않았는지 확인
- marketing_stats_daily와 일치하는지 확인
