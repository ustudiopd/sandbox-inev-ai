# 이메일 발송 설정 가이드

웨비나 등록 후 자동 이메일 발송 기능을 사용하려면 환경 변수를 설정해야 합니다.

## 환경 변수 설정

`.env.local` 파일에 다음 환경 변수를 추가하세요:

```env
# SMTP 설정 (구글 계정)
SMTP_HOST=smtp.gmail.com
SMTP_PORT=587
SMTP_USER=admin@modoolecture.com
SMTP_PASS="앱_비밀번호_여기에_입력"

# 앱 URL (이메일 링크에 사용)
NEXT_PUBLIC_APP_URL=https://your-domain.com
```

**참고:** 
- 값에 공백이나 특수문자가 있으면 반드시 따옴표(`"`)로 묶어주세요
- 구글 앱 비밀번호는 보통 공백 없이 16자리이지만, 안전하게 따옴표로 묶는 것을 권장합니다

## 구글 계정 앱 비밀번호 생성 방법

1. Google 계정 설정 페이지로 이동: https://myaccount.google.com/
2. 보안 탭 클릭
3. "2단계 인증" 활성화 (필수)
4. "앱 비밀번호" 섹션으로 이동
5. "앱 선택" → "메일" 선택
6. "기기 선택" → "기타(맞춤 이름)" 선택 후 "EventLive" 입력
7. 생성된 16자리 앱 비밀번호를 복사
8. `.env.local` 파일의 `SMTP_PASS`에 붙여넣기 (공백이 있으면 따옴표로 묶기)

## 주의사항

- 앱 비밀번호는 일반 비밀번호와 다릅니다
- 2단계 인증이 활성화되어 있어야 앱 비밀번호를 생성할 수 있습니다
- 앱 비밀번호는 한 번만 표시되므로 안전한 곳에 보관하세요
- 환경 변수는 절대 Git에 커밋하지 마세요 (`.env.local`은 `.gitignore`에 포함되어 있습니다)

## 테스트

웨비나 등록 신청 후 등록자에게 자동으로 이메일이 발송됩니다.

이메일 내용:
- 웨비나 제목
- 입장 링크
- 등록 확인 메시지
