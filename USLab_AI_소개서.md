## 🎯 USLab AI는 무엇을 하는가?

**USLab AI**는 **B2B 이벤트 및 마케팅 데이터를 AI로 분석하여 실무자가 바로 실행할 수 있는 인사이트와 액션 플랜을 제공하는 AI 플랫폼**입니다.

복잡한 설문조사 데이터를 단순 통계가 아닌 **"그래서 어쩌라고?"를 해결하는 실행 가능한 의사결정 지원 도구**로 변환합니다.

### 핵심 가치
- **데이터 → 액션**: 통계가 아닌 **실행 가능한 의사결정** 제공
- **AI 기반 자동화**: 설문 응답을 **세일즈/마케팅 플레이북**으로 자동 변환
- **실시간 운영 지원**: 웨비나 운영 중 **Q&A 모더레이션, 채팅 관리** 등 실시간 도구 제공
- **멀티테넌시 SaaS**: 에이전시가 여러 클라이언트를 관리하는 **확장 가능한 플랫폼**

### 1. EventFlow - B2B2C 웨비나 플랫폼

**무엇인가?**
- 에이전시가 여러 클라이언트를 관리하고, 각 클라이언트가 독립적으로 웨비나를 운영할 수 있는 **멀티테넌시 SaaS 플랫폼**
- YouTube 생중계 기반의 **고성능 인터랙티브 웨비나** 제공

**주요 기능**:
- ✅ **웨비나 생성/관리**: 제목, 설명, 일정, 접근 정책 설정 (완료)
- ✅ **실시간 상호작용**: 채팅, Q&A, 설문조사, 퀴즈, 추첨 (완료)
- ✅ **참여자 관리**: 등록, 출석 확인, 역할 관리, 통계 분석 (완료)
- ✅ **운영 콘솔**: Q&A 모더레이션, 채팅 관리, 실시간 모니터링 (완료)
- ✅ **통계 대시보드**: 접속자 수, 참여율, 상호작용 통계, CSV 내보내기 (완료)
- ✅ **화이트레이블**: 클라이언트별 브랜딩 (로고, 색상, 도메인) (완료)
- ✅ **발표자료 관리**: 파일 업로드, 다운로드 추적 (완료)
- ✅ **게스트 모드**: 닉네임만으로 익명 입장 (완료)

**기술 특징**:
- 실시간 데이터 동기화 (Supabase Realtime)
- 멀티테넌시 데이터 격리 (RLS 기반 보안)
- 서버리스 아키텍처 (Vercel + Next.js)

### 2. 설문조사 AI 분석 시스템

**무엇인가?**
- 설문조사 응답 데이터를 **AI가 분석하여 B2B 세일즈/마케팅 의사결정을 지원하는 보고서**를 자동 생성
- 단순 통계가 아닌 **"지금 무엇을 해야 하는가?"**에 대한 구체적인 액션 플랜 제공

**핵심 차별점**:
- ✅ **동적 문항 처리**: 문항 수가 3개든 10개든 자동으로 적응하여 분석
- ✅ **역할 기반 분석**: 문항의 의미(타임라인, 프로젝트 유형, 후속 액션 의향 등)를 자동 추정
- ✅ **Evidence 기반**: 모든 결론이 숫자와 근거로 뒷받침됨
- ✅ **실행 가능한 액션**: 담당자, 기한, 단계별 실행 계획 포함

**생성되는 분석 내용**:

#### Analysis Pack (서버 계산 - 결정론적)
- 문항별 통계 및 분포
- 교차표 분석 (상관관계, Lift 계산)
- 리드 스코어링 (P0~P4 우선순위 분류)
- Evidence Catalog (모든 근거를 ID로 정리)

#### Decision Pack (AI 생성 - 의사결정 지원)
- **Decision Cards (3-5개)**: 핵심 의사결정 질문에 대한 옵션 비교 및 추천
  - "지금 바로 컨택해야 하는 리드는 몇 명인가?"
  - "제한된 영업 리소스를 어느 채널에 배분할까?"
  - "마케팅은 어떤 메시지로 어떤 세그먼트를 먼저 타겟팅할까?"
- **Action Board**: D+0 (24시간), D+7 (7일), D+14 (14일) 단위 실행 계획
- **Playbooks**: 세일즈/마케팅 팀별 구체적인 실행 가이드
- **Survey Next Questions**: 다음 설문에서 추가해야 할 질문 제안

**사용 예시**:
```
입력: 3개 문항 설문조사 (프로젝트 계획 시기, 프로젝트 유형, 후속 액션 의향)
     → 50명 응답

출력:
- P0 리드: 17명 (34%) → 즉시 컨택 필요
- P1 리드: 12명 (24%) → 7일 내 컨택
- 채널 선호도: 방문 요청 45%, 온라인 미팅 30%
- 액션 플랜: D+0에 P0 리드 17명에게 방문 요청, D+7에 P1 리드 12명에게 온라인 미팅 제안
```

**기술 스택**:
- AI 모델: Google Gemini 2.0 Flash
- 분석 파이프라인: Analysis Pack (결정론) → Decision Pack (LLM) → 최종 보고서 (Markdown/PDF)
- 품질 보장: 스키마 검증, 린터, 재시도 로직 (최대 4회)

### 3. 이메일 마케팅 자동화 시스템

**무엇인가?**
- 웨비나/이벤트 관련 **트랜잭션 이메일 및 마케팅 이메일을 자동으로 제작하고 발송**하는 시스템
- 클라이언트가 직접 이메일 초안을 작성하고, 운영자가 승인 후 발송하는 워크플로우

**주요 기능**:
- ✅ **이메일 템플릿 관리**: 리마인드, 등록 확인, 후속 안내 등 유형별 템플릿 (완료)
- ✅ **대시보드 편집**: 클라이언트가 직접 제목/본문 수정 (마크다운 지원) (완료)
- ✅ **AI 기반 초안 생성**: 웨비나/이벤트 정보를 바탕으로 이메일 초안 자동 생성 (완료)
- ✅ **대상자 쿼리**: 등록자 전체, 미입장자, 특정 조건 필터링 (완료)
- ✅ **테스트 발송**: 승인 전 테스트 이메일 발송 (최대 10개) (완료)
- ✅ **실제 발송**: 배치 발송, 동시성 제어, 실패 재시도 (완료)
- ✅ **발송 로그**: 수신자별 발송 상태 추적, 중복 방지 (dedupe_key) (완료)
- ✅ **멀티테넌시 지원**: 클라이언트별 발송 도메인/Reply-To 설정 (완료)
- ✅ **상태 관리**: draft → ready → sending → sent/failed 워크플로우 (완료)

**이메일 유형**:
- 웨비나 등록 확인 이메일
- 웨비나 리마인드 (D-1, H-1)
- 설문조사 참여 안내
- 이벤트 후속 안내

**기술 스택**:
- 이메일 프로바이더: Resend API (프로덕션), Gmail SMTP (레거시)
- 템플릿 처리: Markdown → HTML 변환, 변수 치환
- 발송 방식: 배치 처리, 비동기 큐

### 4. 통계 및 리포트 시스템

**무엇인가?**
- 웨비나 및 이벤트의 **실시간 통계 수집 및 분석 대시보드**
- 관리자가 성과를 한눈에 파악하고 의사결정에 활용

**수집 통계**:

#### 웨비나 통계
- **접속 통계**: 현재 접속자 수, 최대 동시 접속자, 평균 접속자, 시간대별 추이
- **참여 통계**: 등록자 수, 실제 접속자 수, 재입장 비율
- **상호작용 통계**: 채팅 메시지 수, Q&A 질문/답변 수, 설문 제출 수, 추첨 참여 수
- **파일 통계**: 다운로드 수, 파일별 다운로드 통계

#### 설문조사 통계
- 응답 수, 완료율
- 문항별 분포 및 교차 분석
- AI 분석 보고서 (위 "설문조사 AI 분석 시스템" 참조)

**기술 특징**:
- **부하 최소화**: 라이브 참가자 화면에서는 가벼운 ping만 수행
- **서버 크론 기반 집계**: 1분마다 수집, 5분 버킷으로 저장
- **실시간 모니터링**: 관리자 화면에서만 집계 쿼리 실행
